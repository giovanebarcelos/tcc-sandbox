Existem vários sites bem úteis para **testar expressões regulares** online, com destaque para alguns que até mostram explicação detalhada da regex. Eis os principais:  
  
---  
  
## 🔹 Sites recomendados  
  
1. **[Regex101](https://regex101.com/)**  
   
   - Suporte a **PCRE, Python, JavaScript e Golang**.  
   
   - Mostra **explicação detalhada** da regex.  
   
   - Permite salvar e compartilhar exemplos.  
  
2. **[RegExr](https://regexr.com/)**  
   
   - Interface bem intuitiva.  
   
   - Vários exemplos prontos.  
   
   - Ótimo para iniciantes, com **explicação em tempo real**.  
  
3. **[RegexPal](https://www.regexpal.com/)**  
   
   - Simples e rápido.  
   
   - Baseado em JavaScript.  
  
4. **[Regex Tester (PHP)](https://www.phpliveregex.com/)**  
   
   - Voltado para regex em PHP.  
   
   - Mostra grupos e substituições.  
  
5. **[Debuggex](https://www.debuggex.com/)**  
   
   - Mostra um **diagrama visual** da regex.  
   
   - Bom para entender expressões mais complexas.  
  
6. **[RegexPlanet](https://www.regexplanet.com/advanced/java/index.html)**  
   
   - Permite testar em várias linguagens: **Java, JavaScript, .NET, Perl, Python**.  
   
   - Útil para ver diferenças entre implementações.  
  
7. **[Regex101.cn (chinês/inglês)](https://regex101.cn/)**  
   
   - Alternativa ao regex101 clássico.  
   
   - Funciona bem com Unicode.  
  
---  
  
📌 **Dica para ensino:**  
  
- Para iniciantes → **RegExr** (interface amigável).  
  
- Para análise detalhada → **Regex101** (explica cada parte).  
  
- Para visualização → **Debuggex** (gera diagramas).

## 📌 Sugestão de uso didático  
  
- **Primeiras passos**:  
  👉 Usar **RegExr** para que os alunos vejam exemplos e treinem regex simples.  
  
- **Soluções mais intermediárias**:  
  👉 Introduzir **Regex101** para entender grupos, lookahead/lookbehind e explicações passo a passo.  
  
- **Avançado**:  
  👉 Usar no **Debuggex** as regex mais complexas de forma visual.  
  👉 Usar **RegexPlanet** para comparar implementações em linguagens diferentes (Java, Python, etc).
